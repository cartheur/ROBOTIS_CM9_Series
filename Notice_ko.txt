ROBOTIS CM-9시리즈는 하드웨어와 소프트웨어가 모두 공개되는 Open Source입니다. 
최신 정보를 공유하기 위해서는 아래의 링크를 활용하세요.
http://www.robotsource.org/xe/Circle_CM9_Developer_World

CM-9 시리즈는 Arduino와 Maple보드 기반으로 만들어 졌으며 모든 라이선스는 원저작자의 라이선스를 따릅니다.

 1.CM-900 전용 개발 환경인 ROBOTIS CM-9은 Arduino 1.0.1 기반으로 제작되었으며 
  저작권은 Arduino(http://www.arduino.cc/)에 있고 라이선스는 GPL을 따릅니다.

 2. 또한 Arduino는 Processing과 Wiring 프로젝트를 기반으로 하였습니다. Processing과 Wiring 프로젝트에 대해서는
  아래의 사이트에서 정보를 얻을 수 있고 모든 라이선스는 GPL 또는 LGPL 따릅니다.
  http://processing.org/download/


 2.ROBOTIS CM-9에 사용된 ARM 컴파일러는 오픈 소스 GNU 기반의 Code Sourcery G++ Lite for ARM EABI(2010q1-188)
  툴체인을 사용하였습니다. Code Sourcery G++ Lite for ARM EABI는 GPL 또는 BSD 라이선스를 따릅니다.

 3.CM-900의 부트로더는 STMicroelectronics의 stm32f10x_lib 기반으로 제작되었습니다.

 4.ROBOTIS CM-9에서 사용된 API는 Leaflabs의 Maple보드 API Refererence 기반으로 제작되었으며, 
  저작권은 Leaflabs(www.leaflabs.com)에 있으며 라이선스는 MIT License를 따릅니다.
  http://leaflabs.com/licensing/
  http://www.wiring.org.co/download/

 5.ROBOTIS CM-9에서 사용된 다이나믹셀 API는 Dynamixel SDK을 사용해 제작되었으며 
  저작권은 ROBOTIS에 있으며 라이선스는 LGPL를 따릅니다.

 6.CM-900과 ROBOTIS CM-9 프로그램의 시리얼 통신은 rxtx library를 사용합니다. 
  rxtx library의 라이선스는 LGPL을 따르고 아래의 사이트에서 자세한 정보를 얻을 수 있습니다.
  http://rxtx.qbang.org/wiki/index.php/Main_Page
